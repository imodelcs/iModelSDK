# Core Domains 概述

> 原文链接: https://www.itwinjs.org/bis/domains/core/overview/

以下 Domain 构成了 BIS 层次结构中的 [Core layer](../guide/intro/bis-organization.md)。它们定义了"宇宙的结构"（fabric of the universe）和一些关键的组织策略。

## Core Domains 列表

### [Analytical](./analytical.md)

分析领域 Schema，提供专业分析领域 Schema 扩展的 BIS 基础类。

### [BisCore](./biscore.md)

BIS 核心 Schema，包含所有其他领域 Schema 扩展的核心类。

#### 子主题

- [Drawings & Sheets](./drawings-and-sheets.md) - 图纸与图单
- [Provenance in BIS](./provenance-in-bis.md) - BIS 中的溯源

### [Functional](./functional.md)

功能领域 Schema，用于捕获将由 Physical Element 最终实现的功能需求。

### [Generic](./generic.md)

通用领域 Schema，提供通用的建模元素。

### [PhysicalMaterial](./physicalmaterial.md)

物理材料领域 Schema，用于定义物理材料的属性。
