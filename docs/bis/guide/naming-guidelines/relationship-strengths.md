# 标准 Relationship "强度" 名称

| 名称 | 描述 |
| --- | --- |
| Has, References, RefersTo | 暗示 referencing（引用） |
| IsOf | 用于指示类型的实例 |
| Owns, Contains | 暗示 embedding（嵌入） |
